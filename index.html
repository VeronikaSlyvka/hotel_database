Курсова робота

Дана курсова робота присвячена розробці інформаційної системи для управління готелем. Система дозволяє управляти клієнтами, номерами, бронюваннями та рахунками. Проект включає повний цикл роботи: створення, оновлення та видалення записів.

Для зберігання даних використовується база даних MS SQL Server. Серверна частина розроблена на Node.js з використанням Express.js. Фронтенд реалізований на чистому HTML, CSS та JavaScript.
